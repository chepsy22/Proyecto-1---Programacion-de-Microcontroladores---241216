** DESCRIPCION
El repositorio contiene el codigo y la documentacion para un reloj digital en Assembler AVR para el microcontrolador ATmega328p
Se uso una arquitectura con interrupciones y maquina de estados finitos

** CARACTERISTICAS 
Reloj de 24 horas y Calendario
Alarma 
Multiplexado con 4 displays de 7 segmentos
Maquina de estados finitos de 8 modos
3 botones de control
